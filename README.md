# AndroidPhoneCheck（安卓验机）

这是一个用于二手安卓手机检测的手机端 App，支持 Android 9–16。

项目不需要连接电脑，设备信息、相机画面和录音都在手机本地处理。目前主要在 Android 9 的中兴 Blade 20 Smart 上测试，其他品牌和系统版本还需要继续验证。

## 能检查什么

- 手机型号、系统、CPU、内存、存储、电池和屏幕参数
- 屏幕颜色、坏点和多点触控
- 前后摄像头、闪光灯和画面基本情况
- 麦克风录音、扬声器回放、左右声道和扫频
- 音量键和振动
- 光线、距离、加速度、陀螺仪、磁力计等传感器
- Wi-Fi、蓝牙、NFC、定位、SIM 和移动网络状态
- 指纹或面部识别的系统认证
- USB、存储读写和部分安全风险

摄像头和麦克风页面会根据采集到的数据给出辅助提示，例如画面过暗、模糊、偏色，或者录音音量过低、爆音、断音。提示只用来帮助筛查，最终结果仍需要人工确认。

## 隐私

项目没有云端服务。

相机画面只在内存中处理。测试录音保存在 App 的临时目录，退出音频测试后会删除。照片、录音和设备信息不会上传。

## 构建

项目使用 Kotlin、Jetpack Compose、CameraX 和 Gradle，需要 JDK 17 与 Android SDK 36。

```powershell
.\gradlew.bat :app:testDebugUnitTest :app:assembleDebug :app:lintDebug
```

Debug APK 位于：

```text
app/build/outputs/apk/debug/app-debug.apk
```

## 说明

这个项目还在继续测试，不同厂商开放的系统信息和权限并不完全相同。读取不到的项目会显示为不支持或权限受限，不会直接判断为硬件损坏。

检测结果不能用来证明零部件是否原装，也不能代替维修仪器或拆机检查。
